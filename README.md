## Custom Range Counter

A simple JavaScript-based counter application that counts numbers automatically within a custom range entered by the user.

## 🚀 Features

- Counts numbers automatically every second
- Accepts custom start and end values
- Input validation with alert messages
- Responsive UI using Bootstrap
- Beginner-friendly JavaScript project

## 🛠️ Technologies Used

- HTML
- CSS
- Bootstrap
- JavaScript

## 📂 Project Structure

├── index.html  
├── style.css  
└── script.js  

## ▶️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Run the `index.html` file in your browser

## 💡 Concepts Used

- DOM Manipulation
- Event Handling
- setInterval()
- clearInterval()
- Conditional Statements
- Input Validation

## 📸 Output

The user enters:
- From Value
- To Value

After clicking the **Start** button, the counter begins increasing every second until the ending value is reached.
